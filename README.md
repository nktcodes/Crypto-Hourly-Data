# Cryptocurrency Hourly Dataset   

This is a collection of **Hourly** data for a number of cryptocurrency coins.    

This dataset has **1** csv file for each coin.   
   

# Column Information         

* Date    
* Symbol    
* Open    
* High    
* Low    
* Close
* Volume    
* Volume USDT    
* tradecount
